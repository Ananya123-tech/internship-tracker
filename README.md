# Internship Tracker

A web application built using Flask and MongoDB to track internship applications.

## Features
- Add internship details
- View saved internships
- MongoDB database storage
- Modern UI

## Tech Stack
- Python
- Flask
- MongoDB
- HTML & CSS

## How to Run

1. Install dependencies:
pip install flask pymongo

2. Run application:
python app.py

3. Open browser:
http://127.0.0.1:5000
